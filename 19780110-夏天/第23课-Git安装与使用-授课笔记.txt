五、git常规操作
1、向仓库上传新文件/目录
（1）新建一个文件a.txt 或 目录“18302010-李小龙”
（2）添加到git暂存区：git add a.txt 或 18302010-李小龙
（3）提交到仓库：git commit -m "你的说明"

2、查看git状态
 执行命令：git status

3、修改文件
（1）修改文件a.txt内容
（2）添加到git暂存区：git add a.txt
（3）提交到仓库：git commit -m "你的说明"

4、撤销修改（修改仍在工作区，并没有添加到暂存区）
 git checkout -- a.txt

5、回退版本（修改已经添加到暂存区或者仓库了）
 git reset --hard HEAD

6、删除文件，同“3、修改文件”

7、查看日志
 git log
